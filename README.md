
# TravelMate

TravelMate is a group travel planner app for Android and iOS. It helps you organize shared trips, activities, budgets, notifications, and features AI-powered messaging for seamless group coordination.

## Features
- **Group Trips:** Create and manage group trips with shared itineraries, activities, and points of interest.
- **Budget Tracking:** Track group and individual budgets and expenses for each trip.
- **Secure Authentication:** Role-based access for organizers and participants.
- **In-App Chat:** Chat between members plus an AI chatbot for recommendations and quick help.

## Tech Stack
- **Platform:** Native Android
- **Language:** Kotlin
- **Architecture:** MVVM, ViewModel + LiveData/Flow, Repository pattern
- **Backend:** External REST API for auth, trips, activities, budgets, and chat

## Installation
1. Clone the repository:
	```bash
	git clone https://github.com/CodeBloodedAmine/TravelMate.git
	```
2. Open the project in Android Studio.
3. Sync Gradle and build the project.
4. Run on your Android device or emulator.

## Usage
1. Register or log in as a user.
2. Create a new trip or join an existing one.
3. Add activities, manage budgets, and chat with group members.
