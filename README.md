# TravelMate
Group travel planner app for Android and iOS with shared trips, activities, budgets, notifications, and AI-powered messaging

TravelMate
Group travel planner Android app built with Kotlin, featuring shared trips, activities, budgets, notifications, and AI-powered messaging backed by a REST API.​​

Features
Create and manage group trips with shared itineraries, activities, and points of interest.​

Track group and individual budgets and expenses for each trip.​

Secure authentication with role-based access for organizers and participants.​

In-app chat between members plus an AI chatbot for recommendations and quick help.​

Push notifications for upcoming activities, changes, and important updates.​

Tech stack
Platform: Native Android.​

Language: Kotlin.​

Architecture (suggested): MVVM, ViewModel + LiveData/Flow, Repository pattern.​

Networking: Retrofit/OkHttp + Kotlin Coroutines (or your chosen HTTP client).​

Dependency injection: Hilt/Koin (optional but recommended).​

Backend: External REST API for auth, trips, activities, budgets, and chat
